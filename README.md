# 简易前后端分离仓储仓库库存管理系统 java424

## 项目概述

本项目是一个简易的仓储仓库库存管理系统程序，采用前后端分离的技术架构。它适用于初学者交流和学习，提供了用户注册登录、仓库管理、分类管理、物品管理等功能模块。

## 技术栈

- 服务端：Spring Boot
- 客户端：Vue 2 + ElementUI
- 数据库：MySQL
- 运行环境：Node.js、Maven

## 功能模块

- 用户注册登录
- 按权限区分菜单显示
- 动态路由
- 仓库管理
- 分类管理
- 物品管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- IntelliJ IDEA 2021.3.1
- 数据库：MySQL 8.0.34
- Node.js v14.15.4
- Maven 3.5.0

## 部署步骤

1. 准备配置好的Java环境。
2. 安装IntelliJ IDEA工具。
3. 下载Maven包，项目版本为3.5.0。
4. 安装Node.js，项目版本为v14.15.4。
5. 安装MySQL数据库。
6. 使用Navicat Premium数据库操作软件。

### 具体流程

1. 下载项目后，在Navicat中导入sql文件。
2. 在IntelliJ IDEA中分别打开前后端项目。
3. 对后端项目执行`build project`，并使用Maven加载项目依赖。
4. 在前端项目中打开IDEA终端，运行`npm run install`安装依赖。
5. 完成安装后，分别运行前后端项目。

## 目录结构
